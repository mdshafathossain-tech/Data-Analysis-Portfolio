## 📝 Project Overview
This project is designed to automate and streamline the process of tracking, analyzing, and visualizing the attendance and performance of a company's Board of Directors. Built as a **Macro-Enabled Excel (`.xlsm`)** solution, it transforms complex raw attendance logs into a clean, interactive corporate dashboard. 

This project demonstrates the seamless integration of backend data architecture with frontend business intelligence visualization.

---

## 🚀 Key Features

- **🎛️ Interactive Dashboard:** Dynamic Slicers and Timelines allow stakeholders to filter data instantly by Director Name, Quarter, or Year.
- **🔄 Automated Data Aggregation:** Robust Pivot Tables that crunch complex backend metrics into clean, meaningful summaries.
- **🤖 VBA / Macro Automation:** Custom macro automation scripts to clean and refresh data instantly with a single button click.
- **📈 Advanced Visualization:** Modern data charts designed to highlight attendance trends, meeting distributions, and individual participation rates.
- **🛠️ Clean Data Modeling:** Structured tables with normalized data relationships to ensure high performance and scalability.

---

## 🛠️ Tech Stack & Functions Used

### 1. Data Analytics & Visualization:
- **Pivot Tables & Pivot Charts:** For summarizing overall board meeting metrics and individual attendance rates.
- **Slicers & Timelines:** To provide an intuitive, live-filtering user experience.
- **Conditional Formatting:** To visually flag low attendance thresholds (e.g., `< 70%`) requiring immediate board attention.

### 2. Advanced Excel Formulas:
- `XLOOKUP` / `VLOOKUP` (For cross-referencing directors with specific board session IDs).
- `COUNTIF` / `SUMPRODUCT` (For dynamically calculating attendance metrics and participation percentages).
- `IFERROR` (For clean data modeling and error handling).

### 3. Automation Engine:
- **Excel VBA (Visual Basic for Applications):** Custom backend automation that ensures the entire dashboard refreshes flawlessly without manual intervention.
